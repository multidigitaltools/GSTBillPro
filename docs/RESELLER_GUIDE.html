Reseller & White-Label Tutorial

Resell GSTBillPro under your own brand — full step-by-step with screenshots

For Resellers & Partners — v6.0

How it works

Multi Digital Tools
issues you an RK key
→
You (Reseller)
activate, set branding,
generate EK keys
→
Your Customer
installs your-branded app,
activates with an EK key
  • Reseller Key (RK-…) — unlocks your Reseller Tool; sets your key quota & expiry.
  • End-User Key (EK-…) — you generate these for customers (up to quota); hardware-lock & expiry optional.
  • branding.dat — your product name, logo, colours & trial; ship beside the app, no recompiling.

Part A — Vendor issues your Reseller Key

1

Open the Master Key Generator

Multi Digital Tools runs MasterKeyGen.exe (kept private). It opens on the Issue Reseller Key tab.

Master Key Generator — Issue Reseller Key tab
2

Enter reseller name, seat quota & expiry

Type the reseller's name, set the seat quota (how many end-user keys they can create) and the licence expiry. A Reseller ID is auto-generated.

Name "Acme Software Solutions", quota 100, expiry set
3

Generate the RK key

Click Generate RK Key; copy it and send it to the reseller.

Key issued — e.g. RK-HWID1M-0100-… (0100 = 100-key quota)
You send the reseller the RK-… key only. The Master Key Generator stays private with Multi Digital Tools.

Part B — Reseller sets up their brand

4

Open the Reseller Tool

The reseller runs ResellerTool.exe — five tabs: Activation, Branding, Key Generator, Package Builder, Quota.

Reseller Tool — Activation tab (other tabs lock until activated)
5

Enter the Reseller Key

Paste the RK key into Enter Reseller Key.

Reseller key pasted, ready to activate
6

Click Activate

A confirmation shows your Reseller ID, key quota and expiry; the status turns green.

Activation success — ID, quota and expiry confirmed
ACTIVATED — quota remaining shown in green; other tabs unlock
7

Set your white-label branding

On the Branding tab enter your product name, company, support email/phone, website, trial days and logo, then click Save branding.dat.

White-label branding — here rebranded as "QuickBooks Lite" by "Acme Software Solutions"
Result: a branding.dat file — drop it next to the EXE and the app runs fully under your brand.

Part C — Reseller serves customers

8

Generate End-User Keys

On the Key Generator tab set an expiry, optionally tick Hardware Lock, and click Generate Key (each click uses one quota seat). Give one EK key to each customer.

Key Generator — EK-… keys created; quota tracked; Export to CSV
9

Build the distribution package

On the Package Builder tab pick the base EXE + output folder and click Build Package to assemble a your-branded, ready-to-ship folder (EXE + branding.dat + README).

Package Builder — branded, ready-to-distribute folder
10

Track your quota

The Quota tab shows keys used / remaining and your licence expiry any time.

Quota — used, remaining and expiry

Good to know

  • Your quota is enforced — once exhausted, contact Multi Digital Tools to top up.
  • No Visual Studio or recompiling — branding is data-driven (branding.dat).
  • Reports need the Crystal Reports runtime — bundle it + the .rpt files in your installer.
  • The Master Key Generator stays private with the vendor; never ship it.
